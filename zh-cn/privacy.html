隐私政策

Mottainai Pantry 对个人信息和应用数据的处理方式。

本地数据

食材库存、食谱、已保存食谱和设置默认保存在您的设备中。开发者不会收集或保存这些数据。

AI 功能与外部 AI 服务

AI 食谱建议为可选功能。使用时，应用会将生成食谱所需的数据发送给您在设置中选择的外部 AI 服务（OpenAI 或 Google Gemini）：食材名称、类别、数量和单位、到期日、保存位置、开封状态、备注、希望避开的食材、食谱语言以及生成提示词。

请不要在备注或其他字段中填写非必要的个人信息。应用不会为了 AI 食谱建议发送账户信息、支付信息或联系方式；但您在食材名称或备注中输入的内容可能会包含在生成提示词中。

API 密钥与外部服务

API 密钥使用 Android Keystore 中的密钥在设备内加密保存，不会包含在 JSON 备份或应用日志中。它仅用于向所选 AI 服务进行身份验证；开发者不会收集或保存该密钥。AI 使用费用适用相应服务商的定价。

外部 AI 服务对数据的处理适用各自的政策和条款：OpenAI 隐私政策、OpenAI 使用条款、Google 隐私政策、Gemini API 条款。

导出与备份

请自行妥善管理 CSV 导出、导入、备份和恢复文件。
