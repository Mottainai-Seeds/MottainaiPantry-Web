隐私政策

Mottainai Pantry 对个人信息和应用数据的处理方式。

本地数据

食材库存、食谱、已保存食谱和设置默认保存在您的设备中。开发者不会收集或保存这些数据。

AI 服务

使用 AI 功能时，食材名称和其他请求内容可能会使用您输入的 API 密钥发送给所选服务商，例如 OpenAI 或 Google Gemini。

API 密钥

API 密钥通过 Android Keystore 在设备中加密保存。相关服务商的条款、隐私政策和使用费用同时适用。

导出与备份

请自行妥善管理 CSV 导出、导入、备份和恢复文件。

联系我们

如有隐私相关问题，请发送邮件至 mottainaiseeds@gmail.com。
