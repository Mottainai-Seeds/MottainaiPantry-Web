常见问题

关于 Mottainai Pantry、AI 功能、本地数据和计划服务的说明。

使用 AI 功能需要什么？

需要您自己的 OpenAI 或 Google Gemini API 密钥。密钥通过 Android Keystore 加密保存在设备中，AI 服务费用由用户自行承担。

可以使用设备端 AI 吗？

目前尚不支持。封闭测试期间可使用 ChatGPT、Gemini 和模拟服务；无需 API 密钥的设备端 AI 计划在未来更新中提供。

数据保存在哪里？

食材、食谱和设置默认保存在设备本地。目前不提供云同步。

Supporter 和 Premium 可以购买吗？

这两项服务仍在计划中。封闭测试期间尚未实现付费功能，因此目前无法购买。

有 iPhone 版吗？

目前仅提供 Android 版，未来可能考虑 iOS 版。
