# Mottainai Pantry v1.0.3 发布说明

## 概要

这是面向正式发布的候选版本。我们根据封闭测试中的反馈，完善了显示内容和使用说明，并统一了日语、英语和简体中文体验。

## 主要内容

- 整理 AI 食谱推荐、已保存食谱和本地食谱建议的说明
- 统一日语、英语和简体中文的标签与术语
- 更新首页、AI 食谱推荐、照片导入和统计页面截图
- 改善备份恢复、条形码查询和统计功能的说明
- 更新网站语言切换、SEO 信息和支持页面

## 注意事项

目前应用仍处于 Google Play 封闭测试阶段。AI 和食品相关信息仅供参考，食品安全和过敏风险请由用户自行确认。
