# Mottainai Pantry v1.0.3 リリースノート

## 概要

正式公開に向けたRelease Candidateです。クローズドテストで確認した表示と案内を整え、日英中で同じ情報を確認しやすくしました。

## 主な内容

- AIレシピ提案、保存済みレシピ、ローカルレシピ候補の案内を整理
- 日本語・英語・簡体字中国語の表示と用語を整理
- 写真からの食材一括取込と統計画面のスクリーンショットを更新
- バックアップ／リストア、バーコード、統計の案内を改善
- Webサイトの言語切替、SEO、サポート情報を更新

## 注意事項

現在はGoogle Play Closed Test中です。AI提案や食品に関する情報は参考情報であり、食品の安全性やアレルギーについては利用者自身で確認してください。
