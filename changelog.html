Changelog

v1.1.0 Release Candidate

Mottainai Pantry v1.1.0では、ホームを中心に食材管理の流れとバックアップ／リストア確認を安定化しました。

  • ホームを食材管理の中心画面として改善しました。
  • ホームから食材名だけで登録できるよう改善しました。
  • 統計から該当する食材一覧へすぐ移動できるよう改善しました。
  • バックアップ／リストアの検証を安定化しました。

v1.0.3 Release Candidate

Mottainai Pantry v1.0.3では、AIレシピ導線、日英中の表示、Webの言語切替、スクリーンショット、統計、バックアップ案内を整えました。

  • AIレシピの導線を整理し、提案から保存までの流れを改善しました。
  • ChatGPT / Gemini の多言語対応を整理しました。
  • 保存済みレシピ、ローカル候補、避けたい食材の管理を追加しました。
  • レシピ管理の使い勝手を改善しました。
  • バーコード読取、Open Food Facts、上書き確認、エラー表示を改善しました。
  • 統計カテゴリの重複表示を修正しました。
  • バックアップ14日通知を追加しました。
  • 単位順とカテゴリアイコンの表示を改善しました。

v1.0.1 Closed Test Feedback Fix

クローズドテストでいただいたフィードバックをもとに、バックアップ/リストア、買い物リスト、カテゴリ管理、統計表示などを改善しました。

  • バックアップ/リストア時の履歴日付保持を改善しました。
  • AIレシピ生成失敗時のエラー重複表示を修正しました。
  • 買い物リスト導線と重複買い警告を改善しました。
  • 履歴から買い物リストへ追加できるよう改善しました。
  • ユーザー定義カテゴリの追加・編集・削除に対応しました。
  • ユーザー定義カテゴリをバックアップ/リストア対象に追加しました。
  • ピン留め食材の優先表示に対応しました。
  • 統計画面のカテゴリ集計を改善し、「その他」の重複表示を解消しました。
  • 単位表示順とバーコード未検出時の案内を改善しました。

Phase27-0F.5 / Closed Testing readiness

  • Closed Testing向けにローカルAIが開発中であることを確認しました。
  • Supporter / PremiumはComing Soonであり、課金機能は未提供であることを整理しました。

  • 公開前チェックリストとPlay Console向け説明文草案を追加しました。
